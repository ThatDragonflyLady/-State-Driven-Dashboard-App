# -State-Driven-Dashboard-App
assignment-task Dashboard
 I built an assignment dashboard app to track the tasks within an assignment.
The assignments in my classes are not difficult, but there are many tasks to keep track of.
This app will let me:
 input the teachers requirements for the assignment
Submit the required work one requirement at a time
And calculate the percentage of the assignment that is done
And create a check on the task that is done to track the requirements
